# Dark 状态栏 - 酒馆正则替换配置

## 重要说明

**「替换为」必须包在 Markdown 代码块（\`\`\`）内**，否则会显示成代码文本而不是执行。

## 正确配置

### 查找表达式（查找正则）

如果你的状态栏占位符是 `<StatusPlaceHolderImpl/>`：
```
<StatusPlaceHolderImpl/>
```

如果你用的是别的占位符，请改成你实际使用的。  
**不要用** `<UpdateVariable>...</UpdateVariable>` 作为查找——那是变量更新块，替换掉会丢失变量数据。

---

### 替换为（必须完整复制，含首尾反引号）

```
```
<body>
<script>
$(() => {
  const pageUrl = '/dark/index.html';
  const baseUrl = '/dark';
  $.get(pageUrl, (htmlText) => {
    const doc = new DOMParser().parseFromString(htmlText, 'text/html');
    const root = doc.querySelector('#root');
    const rootHtml = root ? root.outerHTML : '<div id="root"></div>';
    const wrap = '<div class="dark-embed-wrap" style="max-width:100%;min-width:0;overflow-x:hidden;width:100%;box-sizing:border-box;">' + rootHtml + '</div>';
    $('body').html(wrap);
    $('body').css({ overflowX: 'hidden', maxWidth: '100%' });
    $('html').css('overflowX', 'hidden');
    $('head').find('link[data-dark-css="1"]').remove();
    $('head').append('<link rel="stylesheet" data-dark-css="1" href="' + baseUrl + '/assets/index.css">');
    const old = document.getElementById('dark-remote-module');
    if (old) old.remove();
    const s = document.createElement('script');
    s.id = 'dark-remote-module';
    s.type = 'module';
    s.src = baseUrl + '/assets/index.js';
    document.body.appendChild(s);
  });
});
</script>
</body>
```
```

---

## 操作步骤

1. **查找**：用 `<StatusPlaceHolderImpl/>`（或你当前的占位符）
2. **替换为**：把上面从第一个 \`\`\` 到最后一个 \`\`\` 的整段复制进去，包括首尾反引号
3. 确保第一条消息里也有对应占位符，例如 `<StatusPlaceHolderImpl/>`

如果之前用的是 `<UpdateVariable>` 做查找，建议改为 `<StatusPlaceHolderImpl/>`，否则会误替换变量更新块。

## 云 + 本地通用版本（跨域）

若你的**云酒馆**和 **Dark** 在同一服务器（如 38.246.237.16），而**本地酒馆**在 127.0.0.1 并跨域加载 Dark：

- 用 `替换为-云+本地通用.txt` 的内容作为「替换为」

**路径说明**：使用 $.get 注入方式，布局更一致（紧靠左侧、边框宽度一致）。baseUrl：
- **同源云酒馆**（38.246.237.16）→ `/dist/Dark`
- **本地/跨域** → `http://38.246.237.16/dist/Dark`

若云酒馆 Dark 在 `public/dark/`，将 `'/dist/Dark'` 改为 `'/dark'` 即可。

同一套正则适用于云端和本地。若加载失败，打开控制台（F12）查看 `[Dark] 加载失败` 报错。
